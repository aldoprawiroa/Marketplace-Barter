# SwapSpace

SwapSpace adalah website simulasi marketplace barter dan tukar tambah barang. Project ini dibuat untuk tugas akhir semester dengan konsep sederhana, interaktif, dan dapat dijalankan langsung melalui GitHub Pages.

Website ini tidak menggunakan backend, database, authentication asli, payment gateway, atau API eksternal. Semua data simulasi disimpan secara lokal di browser menggunakan `localStorage`.

---

## Deskripsi Project

SwapSpace memungkinkan pengguna untuk melihat katalog barang, mencari barang, mengunggah barang milik sendiri, mengajukan penawaran swap, melakukan konfirmasi barter, melihat riwayat transaksi, dan mereset data demo.

Project ini berfokus pada:

- struktur website multi-halaman;
- interaksi pengguna menggunakan JavaScript;
- simulasi data lokal;
- alur marketplace barter;
- tampilan responsif.
